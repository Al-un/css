## Initialise a new framework

```shell
mkdir bootstrap4 && cd bootstrap4
# With Windows PowerShell:
mkdir bootstrap4 | cd
# or
New-Item -ItemType Directory bootstrap4 | Set-Location

yarn init
# answer questions
```
